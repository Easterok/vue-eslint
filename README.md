# @easterok/vue-eslint

```
npm i @easterok/vue-eslint --save-dev
```

Create `eslintrc.js` with content

```
module.exports = {
  root: true,
  extends: ['@easterok/vue-eslint'],
  parserOptions: { tsconfigRootDir: __dirname },
};
```
